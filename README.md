# Swiggy Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the Swiggy Sales dataset. The dashboard provides insights into sales performance, customer behavior, and business trends.

## 🎯 Objectives

- Analyze sales performance
- Monitor revenue trends
- Identify top-performing categories
- Track customer ordering patterns
- Build interactive visualizations

## 🛠️ Tools Used

- Power BI
- Microsoft Excel / CSV
- Data Visualization

## 📂 Dataset

- swiggy.csv

## 📊 Dashboard Features

- Total Sales
- Total Orders
- Revenue Analysis
- Category-wise Sales
- Monthly Sales Trend
- Top Performing Products
- Interactive Filters & Slicers

## 🚀 How to Use

1. Download the repository.
2. Open **swiggy_sales.pbix** using Power BI Desktop.
3. Refresh the data if needed.
4. Explore the interactive dashboard.
